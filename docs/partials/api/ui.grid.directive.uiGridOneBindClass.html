uiGridOneBindClass
directive in module ui.grid

Description

One time binding for the class dom tag. For more information see ui.grid.directive:uiGridOneBind.

Usage

as attribute
<div ui-grid-one-bind-class="{String}"="{Object}"="{Array}">
   ...
</div>

Parameters

ParamTypeDetails
uiGridOneBindClassString

The angular string you want to bind. Does not support interpolation. Don't use {{scopeElt}} instead use scopeElt.

uiGridOneBindClassObject

The object that you want to bind. At least one of the values in the object must be something other than null or undefined for the watcher to be removed. this is to prevent the watcher from being removed before the scope is initialized.

uiGridOneBindClassArray

An array of classes to bind to this element.
