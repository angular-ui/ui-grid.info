uiGridOneBindIdGrid
directive in module ui.grid

Description

One time binding for the id dom tag.

Important Note!

If the id tag passed as a parameter does not contain the grid id as a substring then the directive will search the scope and the parent controller (if it is a uiGridController) for the grid.id value. If this value is found then it is appended to the begining of the id tag. If the grid is not found then the directive throws an error. This is done in order to ensure uniqueness of id tags across the grid. This is to prevent two grids in the same document having duplicate id tags.

Usage

as attribute
<div ui-grid-one-bind-id-grid="{String}">
   ...
</div>

Parameters

ParamTypeDetails
uiGridOneBindIdGridString

The angular string you want to bind. Does not support interpolation. Don't use {{scopeElt}} instead use scopeElt.
