ui.grid.rowEdit

Stable This feature is stable. There should no longer be breaking api changes without a deprecation warning.

This module extends the edit feature to provide tracking and saving of rows of data. The tutorial provides more information on how this feature is best used here.
This feature depends on usage of the ui-grid-edit feature, and also benefits from use of ui-grid-cellNav to provide the full spreadsheet-like editing experience
