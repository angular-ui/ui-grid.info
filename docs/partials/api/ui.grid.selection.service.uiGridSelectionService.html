uiGridSelectionService (service in module ui.grid.selection )

Description

Services for selection features

Methods

  • clearSelectedRows(grid)

    Clears all selected rows

    Parameters

    • grid – {Grid} –

      grid object

  • getSelectedRows(grid)

    Returns all the selected rows

    Parameters

    • grid – {Grid} –

      grid object

  • shiftSelect(grid, clicked, multiSelect)

    selects a group of rows from the last selected row using the shift key

    Parameters

    • grid – {Grid} –

      grid object

    • clicked – {GridRow} –

      row

    • multiSelect – {bool} –

      if false, does nothing this is for multiSelect only

  • toggleRowSelection(grid, row, multiSelect)

    Toggles row as selected or unselected

    Parameters

    • grid – {Grid} –

      grid object

    • row – {GridRow} –

      row to select or deselect

    • multiSelect – {bool} –

      if false, only one row at time can be selected
