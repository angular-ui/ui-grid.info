GridOptions
object in module ui.grid.emptyBaseLayer

Description

GridOptions for emptyBaseLayer feature, these are available to be set using the ui-grid gridOptions

Properties

  • enableEmptyGridBaseLayer

    Enable empty base layer, which shows empty rows as background on the entire grid
    Defaults to true, if the directive is used.
    Set to false either by setting this attribute or passing false to the directive.
