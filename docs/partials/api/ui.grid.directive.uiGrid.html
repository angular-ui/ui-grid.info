uiGrid
directive in module ui.grid

Description

Create a very basic grid.

Usage

as element:
<ui-grid
       ui-grid="{Object}">
</ui-grid>
as attribute
<div ui-grid="{Object}">
   ...
</div>

Parameters

ParamTypeDetails
uiGridObject

Options for the grid to use
