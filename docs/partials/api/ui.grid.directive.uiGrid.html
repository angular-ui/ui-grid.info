uiGrid (directive in module ui.grid )

Description

Create a very basic grid.

Usage

as element:
<ui-grid
       uiGrid="{Object}"
       [external-scopes="{Object}"]>
</ui-grid>
as attribute
<div ui-grid="{Object}"
     [external-scopes="{Object}"]>
   ...
</div>

Parameters

  • uiGrid – {Object} –

    Options for the grid to use

  • external-scopes(optional) – {Object=} –

    Add external-scopes='someScopeObjectYouNeed' attribute so you can access your scopes from within any custom templatedirective. You access by $scope.getExternalScopes() function
