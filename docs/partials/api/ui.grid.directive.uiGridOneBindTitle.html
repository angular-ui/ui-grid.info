uiGridOneBindTitle
directive in module ui.grid

Description

One time binding for the title dom tag. For more information see ui.grid.directive:uiGridOneBind.

Usage

as attribute
<div ui-grid-one-bind-title="{String}">
   ...
</div>

Parameters

ParamTypeDetails
uiGridOneBindTitleString

The angular string you want to bind. Does not support interpolation. Don't use {{scopeElt}} instead use scopeElt.
