uiGridOneBindAriaDescribedby
directive in module ui.grid

Description

One time binding for the aria-describedby dom tag. For more information see ui.grid.directive:uiGridOneBind. 

            <div ng-init="anId = 'gridID32'" ui-grid-one-bind-aria-describedby="anId"></div>
Will become: 
            <div ng-init="anId = 'gridID32'" ui-grid-one-bind-aria-describedby="anId" aria-describedby="gridID32"></div>

Usage

as attribute
<ANY ui-grid-one-bind-aria-describedby="{String}">
   ...
</ANY>

Parameters

ParamTypeDetails
uiGridOneBindAriaDescribedbyString

The angular string you want to bind. Does not support interpolation. Don't use {{scopeElt}} instead use scopeElt.
