GridRow
object in module ui.grid.expandable

Description

Additional properties added to GridRow when using the expandable module

Properties

  • isExpanded

    Whether or not the row is currently expanded.

    Example

    $scope.api.expandable.on.rowExpandedStateChanged($scope, function (row) {
  if (row.isExpanded) {
    //...
  }
});
