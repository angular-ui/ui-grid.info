uiGridOneBindAriaLabelledby
directive in module ui.grid

Description

One time binding for the aria-labelledby dom tag. For more information see ui.grid.directive:uiGridOneBind. 

            <div ng-init="anId = 'gridID32'" ui-grid-one-bind-aria-labelledby="anId"></div>
Will become: 
            <div ng-init="anId = 'gridID32'" ui-grid-one-bind-aria-labelledby="anId" aria-labelledby="gridID32"></div>

Usage

as attribute
<div ui-grid-one-bind-aria-labelledby="{String}">
   ...
</div>

Parameters

ParamTypeDetails
uiGridOneBindAriaLabelledbyString

The angular string you want to bind. Does not support interpolation. Don't use {{scopeElt}} instead use scopeElt.
