uiGridOneBind
directive in module ui.grid

A group of directives that provide a one time bind to a dom element. As one time bindings are not supported in Angular 1.2.* this directive provdes this capability. This is done to reduce the number of watchers on the dom.

Short Example (ui-grid-one-bind-src)

        <div ng-init="imageName = 'myImageDir.jpg'">
          <img ui-grid-one-bind-src="imageName"></img>
        </div>
Will become: 
       <div ng-init="imageName = 'myImageDir.jpg'">
         <img ui-grid-one-bind-src="imageName" src="myImageDir.jpg"></img>
       </div>

Short Example (ui-grid-one-bind-text)

        <div ng-init="text='Add this text'" ui-grid-one-bind-text="text"></div>
Will become: 
   <div ng-init="text='Add this text'" ui-grid-one-bind-text="text">Add this text</div>

Note: This behavior is slightly different for the uiGridOneBindIdGrid and uiGridOneBindAriaLabelledbyGrid directives.
